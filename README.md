Script permettant de convertit des fichiers .mkv directement  en .mp4.
Compatible avec les torrents (Qbittorrent) pour l'automatisation de script vers des dossiers predefini.
Sur Qbitorrent ajouter le script dans : Parametre > Téléchargements et indiquer le chemin du fichier !
  ex : powershell.exe -ExecutionPolicy Bypass -File "C:\Users\%USERNAME%\Documents\Convertisseur\Script_To_Qbitorrent\Script_Converter_mkv.ps1" "%F"
